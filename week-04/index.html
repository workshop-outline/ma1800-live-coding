Week 4 — Positions and Layout

This week we will learn how to control the layout of a webpage by working with positioning and flexbox. These tools allow us to decide exactly where elements appear on the page and how they relate to one another.

With CSS positioning and layout you can:

  • Use relative, absolute, and fixed positioning to place elements.
  • Arrange items in rows or columns with flexbox.
  • Control how nested elements behave inside containers.
  • Organise images into simple grids or layouts.
  • Create structured designs such as a user timeline.

These techniques are the building blocks for more advanced web design and interactive layouts.

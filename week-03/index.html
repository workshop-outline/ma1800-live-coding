Week 3 — Introduction to CSS

CSS (Cascading Style Sheets) is the language used to style HTML webpages. HTML gives a page its structure and content (headings, paragraphs, images, links), while CSS changes how that content looks (colours, fonts, spacing, layout).

With CSS you can:

  • Change text colours, fonts, and sizes.
  • Add backgrounds, borders, and spacing around elements.
  • Arrange elements on the page into different layouts.
  • Create effects and animations (for example, changing colour when you hover over a button).

The term cascading means that if more than one CSS rule applies to the same element, the browser decides which one takes priority — usually the most specific rule, or the last one written.

CSS is always placed inside the <head> of an HTML document, either as:

  • An external file (recommended), linked with:
    <link rel="stylesheet" href="style.css">
  • Or inside the page itself using <style> tags.
